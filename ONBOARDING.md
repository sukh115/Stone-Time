## 30초 요약

**2D 횡스크롤 메트로배니아.** 조약돌 골렘이 돌의 재질을 갈아 끼우며 단단한 적을 뚫고 나아가는 게임. 핵심 하이라이트:

- 키보드 4방향 정밀 조작 (마우스 없음)
- "사륜안" — 시간을 감속해서 전술적으로 판 읽고 타격
- "상대적 강도" — 내 무기 재질 < 상대 재질이면 내가 부서짐
- 흑백 8비트 세상이 특정 지점에서 컬러 16비트로 진화

15분짜리 MVP (수직 절단면)를 목표로 18~28주 안에 완성.

- **아트·레벨디자인 담당** — 비주얼 에셋·맵 레이아웃 초안 (TBD)

### 사운드 담당이라면

**작업 폴더**
- `Assets/Audio/BGM/` — 배경 음악
- `Assets/Audio/SFX/` — 효과음
- `Assets/Data/Materials/MD_*.asset` — `hitSfx` 필드만 채움

**MVP 분량 (예상)**

- 환경 진화 연출 사운드 1종

**툴 세팅**
AI 생성 툴(Suno·ElevenLabs 등) 및 DAW는 Contributor 본인이 준비하는 것을 기본으로 함. 상세 딜리버러블·파일 포맷·협업 플로우는 Notion → `5. 사운드 디자인` 페이지에 정리되어 있음 (담당자 합류 시 Lead가 직접 안내).

## 통합 빌드 주기

- **주 1회** Lead가 통합 빌드 실행 → 올린 에셋이 엔진에 붙음
- 에셋 올린 후 3~5일 안에 빌드에서 확인 가능 (보통)
- Phase 3 이전까지는 Lead가 혼자 코드 뼈대 만들어야 하므로 **에셋 통합은 Phase 3부터 본격**. 그 전엔 더미 에셋으로 돌아감
- 단, **사운드 담당은 Phase 1부터 BGM 톤 탐색·폴리시** 착수 가능 (합류 즉시)
